# A modern and interactive Tic Tac Toe game built with Python and Tkinter, featuring a neon-themed UI, player name input, sound effects, and smooth animations. 

## Features
1.Neon-themed modern UI
2.Player name input screen
3. Start button to begin the game
4. Sound effects
5. Game start sound
6. Win sound
7. Draw sound
8. Animated winning line
9.  Restart game option
10.  Desktop GUI application (no browser needed)

 ## Technologies Used
1. Python 3
2. Tkinter – GUI
3. winsound – Native Windows sound effects
4. No external libraries required
5. Works out-of-the-box on Windows

 ## Game Flow
1. Launch the application
2. Enter Player 1 and Player 2 names
3. Click START GAME
4. Play Tic Tac Toe
5. Enjoy sound feedback and win animations
6. Restart anytime using the Restart Game button
